Como funcionam as regras

As regras são avaliadas automaticamente com base nos dados dos sensores. Quando a condição definida é atingida, o sistema pode executar ações automaticamente.

  • ATIVA → A regra está sendo monitorada e pode disparar ações
  • INATIVA → A regra não é executada pelo sistema
  • Condição → Define quando a regra será disparada
